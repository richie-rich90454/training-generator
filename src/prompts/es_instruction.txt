Tu tarea consiste en transformar el texto proporcionado en una colección completa de pares de preguntas y respuestas (Q&A) detallados que preserven íntegramente todo el contenido informativo del texto original. Este conjunto de datos se utilizará como datos de entrenamiento para IA; por lo tanto, la precisión factual, la integridad, la fidelidad semántica, la coherencia lógica y el valor educativo tienen prioridad absoluta sobre la brevedad. Toda información significativa contenida en el texto original debe estar cubierta por al menos una pregunta. Bajo ninguna circunstancia debe omitirse ninguna información significativa.

1. Lee detenidamente el texto completo antes de comenzar a escribir.

Incluye cada hecho, concepto, argumento, explicación, definición, ejemplo, observación, conclusión, detalle de apoyo, justificación, supuesto, limitación, advertencia, aclaración, excepción, comparación, dependencia, jerarquía, relación, procedimiento, paso, cronología, relación de causa y efecto, detalle contextual, detalle técnico, expresión matemática, fórmula, algoritmo, detalle relacionado con código, contenido de tablas, elemento de listas, cita, referencia, valor numérico, porcentaje, medición, fecha, hora, número de versión, identificador, unidad y nombre propio, cuando corresponda.
4. Genera preguntas claras, autosuficientes y comprensibles cuyas respuestas puedan entenderse sin necesidad de contexto adicional.
5. Toda información significativa del texto original debe aparecer en al menos una respuesta.
6. Siempre que sea apropiado, prioriza respuestas más profundas y completas en lugar de generar numerosas preguntas superficiales.
7. Genera una amplia variedad de tipos de preguntas, incluyendo preguntas factuales, conceptuales, explicativas, analíticas, comparativas, procedimentales, orientadas a la aplicación y de inferencia, siempre que estén directamente respaldadas por el texto original.
8. Si el texto original contiene procedimientos, genera preguntas que cubran cada uno de sus pasos.
9. Si el texto original contiene comparaciones, genera preguntas que cubran todas las similitudes y diferencias.
10. Si el texto original contiene ejemplos, genera preguntas para cada uno de ellos.
11. Si el texto original contiene listas, genera preguntas para cada uno de sus elementos.
12. Conserva la cronología, las jerarquías, las dependencias, los requisitos previos, las clasificaciones, las condiciones y las relaciones de causa y efecto.
13. Las respuestas deben ser completas, técnicamente precisas, internamente coherentes, con una alta densidad de información y basadas exclusivamente en el contenido del texto original.
14. No inventes información, no especules, no infieras hechos no respaldados ni introduzcas conocimientos externos.
15. Genera tantos pares de preguntas y respuestas como sean necesarios para preservar íntegramente el contenido informativo del texto original. No existe un límite máximo para el número de pares de preguntas y respuestas.
16. Mantén una terminología coherente y consistente en todo el conjunto de datos.
17. Todas las preguntas y respuestas deben estar escritas en un español claro, profesional y natural. Salvo los nombres propios, la terminología técnica, el código fuente, las fórmulas, los nombres de archivos o cualquier otro contenido que deba conservarse explícitamente en otro idioma según el texto original, no debe utilizarse ningún otro idioma. Las etiquetas de formato "Question:" y "Answer:" deben permanecer en inglés y no deben traducirse para garantizar la compatibilidad con el formato de salida.
18. Antes de generar el resultado final, realiza una verificación interna completa para asegurarte de que:
   - no se haya omitido ninguna información significativa;
   - no se haya introducido ningún error factual;
   - se haya cubierto cada detalle importante;
   - todos los valores numéricos, nombres, fechas, ejemplos y términos técnicos se conserven con exactitud;
   - cada respuesta esté completamente respaldada por el texto original.
